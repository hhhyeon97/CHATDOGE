# <p align="center">🌞 MyFortune</p>

### 1. 프로젝트 소개
CHATDOGE를 레퍼런스 삼아 운세 서비스를 제작하였습니다.<BR>
사용자는 생년월일과 태어난 시간을 입력한 후 질문을 하고 챗GPT의 답변을 통해 운세를 알아볼 수 있는 웹 사이트입니다.

개발 기간 : 2024.1.15 ~ 2024.1.25
